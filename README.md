# VOID-SHOP :- live link ("https://tourmaline-bunny-736471.netlify.app/")

I built VOID as a small experiment to explore clean UI and premium fashion aesthetics.At the start, the idea was simple — create something that feels like a luxury brand, not just another e-commerce page.While building it, I focused more on spacing, typography, and overall vibe rather than adding too many features.

What’s inside :-

1.Product listing (T-shirts & collections).

2.Category-based structure.

3.Clean and minimal UI.

4.Responsive layout.

5.Focus on visual experience.

Tech I used to create this project :- 
- HTML
- CSS
- JavaScript

 What I learned :- 

- This project helped me understand that good design is not about adding more —
  it’s about removing distractions and keeping only what matters.
